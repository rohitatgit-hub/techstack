Ruby on Rails Stack

Category Technologies
Database Systems PostgreSQL, MySQL, SQLite, MongoDB, Redis (for caching and background jobs)
Frontend Frameworks and Libraries JavaScript (often with jQuery), React.js, Angular.js, Vue.js, Stimulus.js (from the creators of Rails)
CSS Frameworks Bootstrap, Foundation, Bulma, Tailwind CSS
Testing Frameworks RSpec (for behavior-driven development), Capybara (for integration testing), FactoryBot (for creating test data), Selenium (for browser automation)
Deployment and Hosting Heroku, Amazon Web Services (AWS), DigitalOcean, Google Cloud Platform (GCP), Linode
APIs and Web Services RESTful APIs, GraphQL, OAuth (for authentication), JSON Web Tokens (JWT)
Background Job Processing Sidekiq (with Redis), DelayedJob, Resque
Authentication and Authorization Devise, Pundit, CanCanCan
Documentation Swagger, RDoc, YARD
CI/CD Jenkins, Travis CI, CircleCI, GitLab CI/CD
Version Control Systems Git (with GitHub, GitLab, Bitbucket), Subversion (SVN)
Web Servers Apache, Nginx, Puma (default Rails web server), Unicorn
